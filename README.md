# simple-ecommerce-system

## Description

This is a full-stack e-commerce web application built to demonstrate backend development, database design, and core web application functionality. The project focuses primarily on system logic, user authentication, and product management.

## Features

* User registration and login system
* Session management
* Add to cart functionality
* Product quantity handling
* Database integration (CRUD operations)
* Basic admin/product management

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## Project Focus

This project emphasizes backend logic, database operations, and functional e-commerce workflows. The user interface is intentionally kept minimal to prioritize application structure and system functionality.

## How to Run

1. Install XAMPP (or any local server environment).
2. Import the provided database into MySQL.
3. Place the project folder inside the `htdocs` directory.
4. Run the project on `localhost` through your browser.
